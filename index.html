About me

I am a software developer who particularly values clarity, accessibility and sustainable structure in user interfaces. This portfolio is intenionally "framework-free" to make it easy to read and maintain.

I enjoy projects that combine good UX, logical structure and careful implementation, where the user interface is not only "functional" but also pleasant to use. I am particularly interested in front-end-focused projects and roles where I can develop user interfaces for real users and improve existing solutions. I am motivated by seeing how even small details-such as accessibility, clarity and interaction-can genuinely improve a product.
